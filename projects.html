Reverb VST

A reverb VST with 4 differnt types of reverb written on JUCE

Tools used

  • C++
  • JUCE

Objective

Creating my own reverb VST by using 4 different methods, which includes using JUCE reverb module, convolution module with recorded/constructed impulse response, and with multiple delays

The demo section below contains videos that demonstrate how each reverb sound. The audio file is also included in case if video playing is not supported by the browser
